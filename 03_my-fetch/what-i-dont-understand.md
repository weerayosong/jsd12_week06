### 🧬 หมวด OOP (Object-Oriented Programming)

- **`this` คือใคร?:** สับสนว่าคำว่า `this` ชี้ไปที่ตัวแปรไหนกันแน่ ค่ามันเปลี่ยนไปเปลี่ยนมา
    - **💡 คำตอบ:** `this` จะชี้ไปที่ **"คนที่เรียกใช้งานมัน ณ วินาทีนั้น"** ไม่ได้ชี้ไปที่ตัวมันเองเสมอไป
- **สร้าง Class ทำไม?:** มี Object แบบ `{}` อยู่แล้ว ทำไมต้องมาเขียน `class` ให้วุ่นวาย
    - **💡 คำตอบ:** Object `{}` คือของชิ้นเดียว แต่ `class` คือ **"แม่พิมพ์"** เอาไว้ปั๊มของหน้าตาเหมือนกันออกมาทีละหลายๆ ชิ้น (ด้วยคำสั่ง `new`)
- **JS ไม่ใช่ Class แท้ๆ?:** ทำไมเรียน OOP ภาษาอื่นมาแล้วมาเขียน JS มันแปลกๆ
    - **💡 คำตอบ:** เบื้องหลัง JS ใช้ระบบ **Prototype (การสืบทอดสายเลือด)** ส่วนคำว่า `class` เป็นแค่ "ไวยากรณ์แต่งหน้า" (Syntax Sugar) ให้เขียนง่ายคล้ายภาษาอื่นเฉยๆ

### ⏳ หมวด Promise

- **ทำไมโค้ดบรรทัดล่าง รันก่อนบรรทัดบน?:** สับสนที่ผลลัพธ์จาก Promise ดันโผล่มาทีหลังโค้ดที่อยู่ด้านล่างมัน
    - **💡 คำตอบ:** Promise คือระบบ **"เดี๋ยวทำให้นะ ไปทำอย่างอื่นก่อนเลย"** โค้ดหลักจึงไม่รอและวิ่งข้ามไปทำบรรทัดล่างสุดทันที
- **แกะของออกจาก Promise ไม่ได้:** พยายามเอาตัวแปรไปรับค่าจากฟังก์ชัน Promise ตรงๆ (เช่น `let data = getPromise()`) แล้วได้คำว่า `Promise {<pending>}`
    - **💡 คำตอบ:** ข้อมูลมัน "ยังมาไม่ถึง" ต้องไปยืนรอรับของในท่อ **`.then(data => ...)`** หรือใช้ `await` เท่านั้น
- **ต่อ `.then()` ยาวๆ แล้วข้อมูลหาย:** ทำเชน `.then().then().then()` แล้วบรรทัดที่สองบอกว่า `undefined`
    - **💡 คำตอบ:** ใน `.then()` ตัวแรก คุณลืมเขียนคำว่า **`return`** เพื่อส่งของไม้ผลัดไปให้ `.then()` ตัวถัดไป

### 🌐 หมวด Fetch API

- **ทำไมต้องมี `.then()` ถึง 2 รอบ?:** โค้ดชุดที่แล้วที่คุณเพิ่งเรียนไป ดึง API ทีไรต้อง `.then(res => res.json())` แล้วถึงจะ `.then(data => ...)`
    - **💡 คำตอบ:** รอบแรกคือการ **"รับกล่องพัสดุ (Response)"** รอบสองคือการ **"กรีดกล่องเอาของข้างในออกมา (JSON)"**
- **Error 404 / 500 แต่ทำไมไม่ตกไปที่ `.catch()`?:** พิมพ์ URL ผิดจนหน้าเว็บพัง แต่โค้ดดันวิ่งเข้า `.then()` หน้าตาเฉย
    - **💡 คำตอบ:** Fetch จะโยนเข้า `.catch()` **เฉพาะตอนที่ "เน็ตหลุด" เท่านั้น** ถ้าเซิร์ฟเวอร์ตอบกลับมา (แม้จะด่าว่า 404 Not Found) ถือว่าทำสำเร็จ! เราต้องเขียนเช็ค `if (!res.ok)` เอาเอง
- **CORS Error สีแดงเถือกเต็มจอ:** ดึง API แล้วติด Error ว่า _No 'Access-Control-Allow-Origin'_
    - **💡 คำตอบ:** เป็นระบบความปลอดภัยของ Browser **"เซิร์ฟเวอร์ปลายทางไม่อนุญาตให้หน้าเว็บเราแอบไปดูข้อมูลของเขา"** (แก้ที่ฝั่งเราไม่ได้ ต้องให้ฝั่งเซิร์ฟเวอร์เปิดทางให้)
